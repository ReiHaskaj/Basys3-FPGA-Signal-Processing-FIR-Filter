# FPGA FIR Filter (Basys3)

This project implements a fixed-point FIR filter on FPGA, starting from MATLAB ending with the RTL implementation.

## Features
- FIR filter design in MATLAB
- Fixed-point quantization (Q format)
- Bit-accurate simulation
- Planned FPGA implementation (Basys3)

## Learning Goals
- Understand FIR filtering in time domain
- Analyze quantization effects
- Map DSP algorithms to hardware

## Tools
- MATLAB
- Xilinx Vivado (planned)
